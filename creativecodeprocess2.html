about

Problem

Designers who code and developers who design need a space to learn, create and share work, and collaborate on skills and projects.

Solution

An online community for web designers and developers where they can create and share work, read articles, apply for jobs, and more. This platform consists of an online code editor and a Sketch file previewer, making it useful for those who design, those who code, and those who do both.

Competitive analysis

Codepen

Codepen is a social development environment for front-end developers that includes an inline code editor and is open source.

Sketchviewer

Sketchviewer is an online platform for sharing design work by allowing the previewing of Sketch files and providing version control.

GitHub

GitHub is a Git repository hosting service that provides a web-based graphical interface.

Opportunity: Providing a service that caters to both designers and developers, provides a sense of community, and has personality.
