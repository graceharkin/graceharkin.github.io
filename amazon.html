Amazon Mobile

brand identity // print // packaging // publication

This redesign concept consisted of identifying areas in the current Amazon mobile app to change in order to improve the user experience. I chose to focus on the tasks of filtering, searching, and changing delivery dates. This project went through multiple redesigns as I conducted user testing.

The redesign consisted on a using a dark theme color palette in order to create a modern, hi-tech look and feel to the app. I also focused on making the feed content more intentional and geard towards the particular user. This meant most of the suggestions were based on their previous purchases, and they were shown less options in order to not overwhelm the user.
